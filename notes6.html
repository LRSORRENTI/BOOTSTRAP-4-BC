Carousels

The final component we'll be going over in the course are carousels, they're used to make image galleries, sliders, or carousels

We can set the slider to change at timed intervals, like every 5 seconds, 10, or however many seconds we want

We can also set arrow buttons to manually change betweeen slides

One thing to keep in mind, carousels aren't very well-adapted to accessibility, they're not easy for people who're using screen readers, putting things that are vital to the website inside the carousel is not recommended and shouldn't be done

Carousels are mostly meant for decoration purposes

Also note that we must include '.d-block' and '.v-100' on carousel images to prevent browser default image alignment

Also note if we don't put our carousel inside of a container div, the carousel itself will go across the entire page

Note that we must always include < div class="carousel item active" to our first carousel item, if we omit this, nothing will display

To change the interval between slide changes, we do that on the carousel, after 'data-ride=carousel' we add 'data-interval=1000', 1000 milliseconds is 1 second so it will change much faster
