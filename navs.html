So if we inspect our nav in the browser, you'll see that display is set to flex, if we remove display:flex, the nav will default to column, and if we add it back our nav goes back to row

In essence flexbox is a box of space, and we can control where our elements go, right now they're all at the start, but we could push them to the end of our flex container, or center them inside our flex container, spread them out

It's important to remeber, navs (nav class = ) are auto-set to flexbox, if we used a div or any other element, we have to specify flex with ( d-flex, d-inline-flex, d-sm-flex, d-md-flex, d-lg-flex, d-xl-inline-flex etc..)

The docs are extemely good at illustrating how it's all set up, always reference the official bootstrap docs when needed, and keep in mind what version of bootstrap docs you're looking at, make sure is you're using bootstrap 4, that you're not looking at the Bootstrap 3 docs.
