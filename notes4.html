Collapse

Collapse isn't exactly it's own component, rather it's a JavaScript plug-in

We add a few pieces to existing components, to show / hide content

Essentially it's a way of connecting a button or a link to another peice of content you want to display

To start we need to make a button or a link, then we set the 'data-toggle="collapse"' inside or anchor or button tag, then we specify an href="#paragraphOne" or whatever html content, but we need to remember to give the element the id, then link that id in our div / button / anchor tag
